# Media Framing of Palestine – CS 418 Project

## Setup Instructions

### 1. Clone the Repository
git clone https://github.com/SamiFariz/MediaFramingOfPalestine.git
cd MediaFramingOfPalestine

### 2. Create Virtual Environment
python -m venv venv

### 3. Activate Environment
Windows - venv\Scripts\activate
Mac - source venv/bin/activate

### 4. Install Required Packages
pip install pandas numpy scikit-learn matplotlib nltk textblob requests jupyter

### 5. Run Cells / Command
run - python
then run the cell :
import nltk
nltk.download('stopwords')

from textblob import download_corpora
download_corpora()

exit()

### 6. Launch the Project
jupyter notebook

### 7. Open the Notebook
Select the .ipynb file called progress_report
